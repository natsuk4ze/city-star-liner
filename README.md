# City Star Liner🌌

Hi👋 City Star Liner is 3D shooting game created by the team _RedCopper👒_(members below) at [HAL Tokyo🏫](https://www.hal.ac.jp/tokyo) using DirectX.
Draw the trail and defeat the enemy.
